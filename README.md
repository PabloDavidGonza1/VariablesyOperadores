Respuestas Teóricas

¿Para qué se utilizan los operadores lógicos en programación?
Los operadores lógicos se utilizan para evaluar condiciones y tomar decisiones dentro de un programa. Permiten combinar comparaciones y controlar el flujo del programa según se cumplan o no ciertas condiciones.

¿Por qué es importante declarar correctamente el tipo de dato de una variable?
Porque el tipo de dato define qué información puede almacenar una variable y qué operaciones se pueden realizar con ella. Declarar correctamente evita errores y mejora el funcionamiento del programa.

Justificación Técnica

Clasificación de edad
Utilicé if - else if - else porque la edad se evalúa por rangos numéricos y esta estructura permite comparar condiciones de forma ordenada.

Día de la semana
Utilicé switch porque se trabaja con un solo valor entero y varias opciones fijas, lo que hace el código más claro y fácil de leer.

Verificación de acceso
Utilicé if - else porque se necesitan evaluar varias condiciones lógicas relacionadas con el usuario y la contraseña.
